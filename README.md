# Maphophe Community System

A platform designed to support digital governance, communication, and service delivery for local communities.

## 🚀 Overview
This system enables villages to manage announcements, service requests, and community engagement digitally.

## 🧠 Current Status
🚧 MVP in development

## 🏗️ Planned Features
- Community announcements
- Service request tracking
- Voting and feedback system

## ⚙️ Tech Stack
- Next.js
- TypeScript
- PostgreSQL
- Prisma

## 📄 Notes
This project is under active development.
